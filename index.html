WeHelp 申請者 - 林岳瑾

個人簡介

十年英語教學，七年資安工作：跨領域價值創造者
從翻譯、業務行銷，到 Python 流程優化，CMS 網站開發，GitHub Hugo 靜態網站開發，在科技公司，文組生進入半資訊工程的世界。體驗到科技的力量後，將 Python、Heroku、Dialogflow、LINE 聊天機器人等工具帶入教學。

洞見市場需求 決定自己寫程式
2018年進入師大研究所進修，發現英語老師仍使用耗費時間的方式查詢資料、製作講義，於是，我去年為一百多位英語老師舉辦 Python 工作坊，並和指導教授一同在師大研究所開課，分享使用 Python 優化備課流程。現在，我決定要創造更多網頁應用，幫助台灣教育工作者。

2017年用 TK UI 工具，結合 Python，做了一個字典速查工具給台灣的翻譯者。Link to the source code

2018、2019年，協助師大研究所教授，用 Python 串接 Pandorabot 聊天機器人、開發 Google Assistant App、串接 LINE 聊天機器人、串接 Google 文字轉語音服務、Google 翻譯服務等。專案紀錄連結

2021年自己辦 Python Workshop 給教育工作者，並開始在研究所開班授課，針對英語老師的需求，以 Colab 為平台，開發流程優化工具給英語老師，並以此為畢業論文。Colab 平台程式碼

為了將寫好的 Python 腳本呈現在網頁，讓更多人能直接操作使用，開始學習網站架設 Coursera 課程 (html, css, javascript)，目前剩 Javascript 還沒學。

時間安排

目前還有資安公司的正職工作，預計九月底離職，十月初全心專注投入在 WeHelp 的課程，經濟收入則靠傍晚在補習班教書，維持一定收入。預計以「實作專案」為主要目標，以最終作品為學習考量，因此可以接受投入假日的時間。

軟體公司

轉職成為軟體工程師後，預計和老婆在英國生活。目前老婆正在英國工作，軟體公司的選擇，目前以在倫敦的資安公司 Callsign、Onfido 為主，持續累積在資安領域的技能和知識。

克服的難題

人生中最大的難題，就是和老婆的溝通。她接受西式教育，很有自己的主見，送禮不合心意，就等著被霸凌，講話沒重點，等著被批鬥。因此，我只好不斷 Try .. Except..，每天記錄溝通不順暢的點，更深入思考她講每一句話的語意，彷彿在寫 API 一般....。現在心靈相通，比較能在同個頻率溝通。過程中領悟到：和老婆溝通就像寫程式，遇到挫折不要放棄，不要難過，看到 Traceback 是基本，不知道正確答案的話，Loop 給它開下去，總會看到希望之光。

網頁開發心得

第一次獻給此網站
目前雖然學完 html 和 css 的知識，但還沒實務開發過，因此這是開發網站的寶貴經驗。有了這次的經驗，即便未來會接觸到更深的內容，也能回想開發這網站的過程，一步一步來，先定位目標，釐清問題，尋找解方，慢慢地就可以做出心目中的網站。

若用想的，就很困難，捲起袖子，先寫就對了
雖然說寫作文、寫程式都必須擬好框架，但當技術門檻成為一種恐懼和阻礙時，先不論框架，從最基本的開始嘗試。先從 heading、paragraph 標籤開始，填入內容，接著思考字體，然後再去調整顏色，不要好高鶩遠，一步一步慢慢來，才能克服恐懼，架好基本網站。

用過 Hugo，但還沒自己架設 from scratch
之前使用過 hugo 模板在 GitHub 架設網站，但尚未從無到有創造出網站框架，因此這次是非常難忘的經驗。

其他想說的

只能說非常感謝 WeHelp 團隊，在我人生中的重大轉折，思考轉職方向時，剛好看到 WeHelp 的課程，這真的是天降甘霖，真心希望能不辜負 WeHelp 團隊的辛勞，具備前後端的基本開發技能，實現去英國和老婆一起生活的願望，並且開發實用網頁型的備課工具，讓英文老師不要再那麼辛苦。
